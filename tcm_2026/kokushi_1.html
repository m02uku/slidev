言語記号の特性

ソシュールの言語観

恣意性

  • 音と意味の結びつきには必然性がない
  • 同じ意味でも言語によって音が異なる

例

「🐶」= /inu/（日）= /dog/（英）

線状性

  • 音声は時間軸上に一次元的に配列される
  • 同時に複数の語（音）を発音できない
  • 文は一定の順序で配列される

重要

  • 言語の恣意性は、言語が社会的な約束事であることを示す

シニフィアンを「記号表現」や「能記」[1]、シニフィエを「記号内容」や「所記」などと訳す

二重分節

言語の経済性を支える仕組み


文\(\qquad\)—\(\qquad\)単語\(\qquad\)—\(\qquad\)形態素\(\qquad\)—\(\qquad\)音素

\(\qquad\qquad\qquad\quad\)(第一次分節)\(\qquad\qquad\quad\quad\)(第二次分節)


第一次分節（形態的分節）

  • 意味を持つ最小単位に分ける

例

  • 「読まない」\(\rightarrow\)「読ま」+「ない」
  • 形態素レベルの分割

第二次分節（音韻的分節）

  • 意味を持たない音の単位に分ける

例

  • 「ネコ」\(\rightarrow\)/n/+/e/+/k/+/o/
  • 音素レベルの分割

音素

意味の違いをもたらす最小の音の単位


音素を見つける方法

最小対（ミニマルペア）

日本語の例

  • 「サカナ」vs 「タカナ」
    • /s/ と /t/ は異なる音素
  • 「カキ」vs 「カギ」
    • /k/ と /g/ は異なる音素
  • cf.「ハシ（橋）」vs 「ハシ（箸）」
    • アクセントも音韻的要素

英語の例

  • “bat” vs “pat”
    • /b/ と /p/ は異なる音素
  • “ship” vs “sheep”
    • /ɪ/ と /iː/ は異なる音素

異音

同じ音素の異なる音声的実現形


相補分布

異音が現れる環境が重ならない

例：日本語の /g/

語頭 → [g]（破裂音）：「学校」[gakkoː]
語中 → [ŋ]（鼻濁音）：「午後」[goŋo]

自由変異

同じ環境で交換可能な音の違い

例

「数学」の「す」→ [sɯ] (非円唇) または [su] (円唇)

弁別的素性

音素を区別する音声的性質


特徴 例：日本語 対立
声帯振動 /k/ vs /g/ 無声 vs 有声
調音位置 /t/ vs /k/ 歯茎 vs 軟口蓋
調音方法 /t/ vs /s/ 破裂 vs 摩擦
鼻音性 /ŋ/ vs /g/ 鼻音 vs 口音


重要

とある素性が弁別的であるかは言語によって異なる

  • 英語：帯気性の有無は「余剰的素性」（ミニマルペアがない）
  • クメール語：帯気性の有無で意味の変わる単語がある（ミニマルペアがある）

休憩

後半は口頭試問

問題 次のうち、言語の恣意性の説明として正しいものはどれか。

  1. 音と意味の結びつきには必然性がある
  2. 音と意味の結びつきには必然性がない
  3. すべての言語で同じ音は同じ意味を持つ
  4. 音声記号は世界共通である
  5. 言語は本能的に獲得される

言語の恣意性：音と意味の結びつきが恣意的であり、必然性がないという原理。

  • 同じ意味でも言語によって異なる音形を持つ
    • 「犬」= /inu/（日本語）、/dog/（英語）、/chien/（フランス語）
  • これは言語が社会的な約束事であることを示す
  • ソシュールが提唱した言語学の基本原理

よくある誤解： 擬音語（オノマトペ）は音と意味に関連があるように見えるが、これも言語によって異なる（日本語「ワンワン」vs 英語「bow-wow」）

  • ソシュールの『一般言語学講義』が出典
  • 恣意性は言語記号の本質的特徴の一つ
  • ら抜き言葉：可能の助動詞「（ら）れる」を「-areru」とすると、

問題 「食べられない」という語を二重分節の観点から分析した場合、第二次分節の結果として最も適切なものはどれか。

  1. /t/ + /a/ + /b/ + /e/ + /r/ + /a/ + /r/ + /e/ + /n/ + /a/ + /i/
  2. 食べ + られ + ない
  3. 食べられ + ない
  4. 食べら + れない
  5. 分節できない
  • 第一次分節（＝単語または形態素）
    • 「食べられない」→「食べ」+「られ」+「ない」
  • 第二次分節（＝音素）
    • 「話せない」→ /h/ + /a/ + /n/ + /a/ + /s/ + /e/ + /n/ + /a/ + /i/
  • 二重分節により、少数の要素で無数の意味を表現できる（言語の経済性）
  • マルティネの用語
    • 第一次分節＝monème 記号素（典型的には「語」）
    • 第二次分節＝音素: 弁別的な最小単位
  • 言語の重要な特徴
  • 言語の経済性: 最小限の労力（発音や文法構造）で最大限の情報（意味）を伝達しようとする言語の働き・原理

問題 次のうち、日本語における最小対（ミニマルペア）の例として正しいものはどれか。

  1. 「橋」と「箸」
  2. 「雨」と「飴」
  3. 「サカナ」と「タカナ」
  4. 「カキ」と「カギ」
  5. 上記すべて

最小対とは、一つの音素（または韻律的要素）だけが異なる語の組

  1. 「橋」[hashi’] と「箸」[ha’shi] アクセントの違い
  2. 「雨」[a’me] と「飴」[ame] アクセントの違い
  3. 「サカナ」/sakana/ と「タカナ」/takana/ 音素 /s/ と /t/ の対立
  4. 「カキ」/kaki/ と「カギ」/kagi/ 音素 /k/ と /g/ の対立（無声 vs 有声）
  • 最小対は音素を特定する基本的手法
  • アクセントも日本語では音韻的に重要

問題 日本語東京方言（標準語）における「ガ行」の子音 /g/ について、次のうち正しい説明はどれか。

  1. 通時的に、すべての位置で常に破裂音[g]として発音される
  2. 語頭では破裂音[g]、語中では鼻音[ŋ]となることがある
  3. 語頭でも語中でも常に鼻音[ŋ]として発音される
  4. /g/と/ŋ/は日本語で異なる音素である
  5. 位置による発音の違いはない

日本語の /g/ は相補分布にある異音を持つ：

  • 語頭：破裂音 [g] 例：「学校」[gakkoː]、「元気」[geŋki]
  • 語中：鼻音 [ŋ]（鼻濁音） 例：「午後」[goŋo]、「刺激」[ɕiŋeki]
  • 相補分布 = 異音が現れる環境が重ならない
  • [g]と[ŋ]は同一音素 /g/ の異音
  • ただし、現代日本語では鼻濁音の使用が減少傾向
  • 東京方言などで観察される現象
  • 若年層では鼻濁音が消失しつつある
  • ガ行鼻濁音は、東北方言では語中のガ行音素（濁音音素）/ɡ/そのものとして現れる。東北方言では、カ行（清音）子音/k/は語頭では[k]だが、ダ行と同様に、語中では有声音化して[ɡ]となるため、正真正銘のガ行子音（濁音）/ɡ/は、語中ではすべて鼻濁音化して[ŋ]と発音され、清濁の弁別が保たれることになる。
    • つまり、東北方言では、基底では/k/と/g/は弁別的だが、表層では[k][g]と[ŋ]が弁別的

問題 「新聞」という語が[ɕimbuɴ]のように発音される現象を説明する音韻規則として正しいものはどれか。

  1. 異化
  2. 同化
  3. 脱落
  4. 挿入
  5. メタセシス（音位転換）
  • /ɴ/ → [m] の変化 /ɕiɴ+buɴ/ → [ɕimbuɴ]
  • 後続の /b/（両唇音）の影響で /n/（歯茎鼻音）が [m]（両唇鼻音）に変化

同化の種類

  • 順行同化：前の音の影響を受ける
  • 逆行同化：後ろの音の影響を受ける
  • 調音の労力を減らす自然な変化
  • 日本語では一般的な音韻現象
  • メタセシス：
    • 新しい: アラタシ → アタラシイ
    • 山茶花: サンザカ → サザンカ
    • 舌鼓: シタツヅミ → シタヅツミ
    • 秋葉原: アキバハラ → アキハバラ

問題 「学校」という語の音節数とモーラ数の組み合わせとして正しいものはどれか。

  1. 音節2、モーラ2
  2. 音節2、モーラ3
  3. 音節2、モーラ4
  4. 音節3、モーラ4
  5. 音節4、モーラ4

「学校」/gakkoː/

  • 音節数：2音節 gak.koː
  • モーラ数：4モーラ が・っ・こ・う
  • 促音「っ」、長音「ー」、撥音「ん」はそれぞれ1モーラ
  • 日本語はモーラ言語（各モーラが等時性を持つ）
  • 日本語がモーラ言語というのは、音声的分析によって示される
  • 俳句の五七五もモーラ単位（モーラがリズム単位＝モーラが等時性を持つ）

問題 音素の定義として最も適切なものはどれか。

  1. 音声の物理的性質の最小単位
  2. 意味の違いをもたらす最小の音の単位
  3. 意味を持つ最小の単位
  4. 聞き分けられる最小の音の単位
  5. 文字で表される最小の音の単位

音素の定義：

  • 意味の違いをもたらす最小の音の単位 （弁別的な最小の単位）
  • 音声の機能的な単位（物理的な単位ではない）
  • 最小対によって特定できる

例

  • 「サカナ」と「タカナ」→ /s/と/t/は異なる音素
  • 「カキ」と「カギ」→ /k/と/g/は異なる音素
  • 音素は抽象的な単位
  • 音声学の「音」と音韻論の「音素」は異なる

問題 異音の説明として正しいものはどれか。

  1. 異なる音素のこと
  2. 同じ音素の異なる音声的実現形
  3. 意味の異なる音のこと
  4. 外来語に特有の音のこと
  5. 方言による音の違い

異音の特徴：

  • 同一音素の環境による変異形 意味の違いをもたらさない
  • 相補分布または自由変異の関係にある

例：日本語の /t/

  • /a e o/ の前：[t]「たばこ」[tabako]
  • /i/ の前：[tɕ]「ちから」[tɕikara] → これらは同一音素 /t/ の異音
  • /u/ の前：[ts]「つくえ」[tsukue]
  • 話者は通常異音の違いを意識しない
  • 音韻論の重要概念

問題 次のうち、弁別的素性に関する説明として正しいものはどれか。

  1. すべての言語で同じ特徴が弁別的である
  2. 言語によって弁別的な素性は異なる
  3. 弁別的素性は音素とは無関係である
  4. 弁別的素性は意味に関わらない
  5. 弁別的素性は音の高さのみを指す

弁別的素性：音素を区別する音声的性質 言語によって何が弁別的かは異なる

重要

とある素性が弁別的であるかは言語によって異なる

  • 英語：帯気性の有無は「余剰的素性」（ミニマルペアがない）
  • クメール語：帯気性の有無で意味の変わる単語がある（ミニマルペアがある）
  • ヤコブソンの弁別的素性理論
  • 音韻体系の記述に重要
  • 音素を区別する＝ミニマルペアがある

問題 相補分布の説明として正しいものはどれか。

  1. 音素が自由に交換可能な状態
  2. 異音が現れる環境が重ならない状態
  3. 音素が意味を区別する状態
  4. 音が連続して現れる状態
  5. 音の配列に制約がない状態

相補分布の特徴：

  • 異なる音が現れる環境が互いに排他的
  • 相補分布にある音は同一音素の異音 環境によって出現が予測可能

例：日本語の /h/

  • /a e o/ の前：[h]「はな」[hana]
  • /i/ の前：[ç]「ひと」[çito] [h]、[ç]、[ɸ]は相補分布にあり、同一音素/h/の異音
  • /u/ の前：[ɸ]「ふね」[ɸɯne]
  • 異音を特定する重要な基準
  • 最小対と対照的な概念
  • h: 無声正門摩擦音
  • ç: 無声硬口蓋摩擦音
  • ɸ: 無声両唇摩擦音

問題 自由変異の説明として最も適切なものはどれか。

  1. 音素が自由に変化すること
  2. 同じ環境で交換可能な音の違い
  3. 意味が変わらずに語順が変わること
  4. 方言による音の違い
  5. 音韻規則が適用されないこと

自由変異の特徴：

  • 同じ環境で複数の音が交換可能
  • 意味の違いをもたらさない
  • 話者や発話スタイルによる変異

相補分布との違い：

  • 相補分布：環境によって決まる
  • 自由変異：環境が同じでも異なる音が現れる

例

  • 「数学」の「す」→ [sɯ] または [su]
  • 個人差や文体差の要因
  • 音韻変化の萌芽となることもある

問題 音節の定義として最も適切なものはどれか。

  1. 文字の最小単位
  2. 発音の自然な単位
  3. 意味の最小単位
  4. 音素の組み合わせのすべて
  5. 時間的に等しい単位

音節の特徴：

  • 発音の自然なまとまり
  • 通常、母音を核とする
  • リズムや韻律の基本単位になりうる

音節の構成：

  • 頭子音（onset）：音節の始まり
  • 核（nucleus）：音節の中心
  • 尾子音（coda）：音節の終わり

例

「桜」= /sa.ku.ra/（３音節）

  • 音節は言語によって構造が異なる
  • 日本語は比較的単純な音節構造

問題 モーラ（拍）に関する説明として正しいものはどれか。

  1. 音節と同じ単位である
  2. すべての言語に存在する単位である
  3. 日本語のリズムの基本単位である
  4. 母音のみで構成される
  5. 意味を持つ最小単位である

モーラ（拍）の特徴：

  • 日本語のリズムの単位
  • 時間的に等しい長さを持つ
  • 日本語はモーラ言語

例：「新聞紙」

  • 音節：shin.bun.shi（3音節）
  • モーラ：し・ん・ぶ・ん・し（5モーラ） 促音、撥音、長音は特殊モーラ
  • 英語は強勢言語（モーラ言語ではない）
  • 俳句の五七五はモーラ単位

問題 音韻規則の「脱落」の例として正しいものはどれか。

  1. 「新聞」/sinbun/ → [ɕimbun]
  2. 「雰囲気」→「ふいんき」
  3. 「です」/desu/ → [des]
  4. 「学校」/gakkou/ → [gakkoː]
  5. 「日本」/nihon/ → [nippon]

  • 「です」：語末の/u/が脱落して[des]

  • 高速発話でよく起こる
  • 他の選択肢

問題 「っ」（促音）に関する説明として正しいものはどれか。

  1. 音節を構成する
  2. 1モーラとして数えられる
  3. 2モーラとして数えられる
  4. すべての子音の前に現れる
  5. 音素/t/として分析される

促音「っ」の特徴：

  • 1モーラとして数えられる
  • 後続子音の閉鎖（つまり）を表す 音節の韻尾（coda）を形成
  • 音声的実現：後続子音と同じ調音位置での閉鎖

例

  • 「学校」：が・っ・こ・う（4モーラ）
  • 「切手」：き・っ・て（3モーラ）
  • 音韻論的には特殊モーラ
  • 音節末子音の一種
  • /p, t, k, s, ɕ/の前に現れる
  • 有声音の前には通常現れない（「っが」は稀）

問題 「ん」（撥音）に関する説明として正しいものはどれか。

  1. 常に[n]と発音される
  2. 後続音によって調音位置が変わる
  3. 母音である
  4. 音節の頭に立つ
  5. モーラとして数えない

撥音「ん」の特徴：

  • 1モーラとして数えられる
  • 後続音によって異音がある

異音の例

  • [m]：/p, b, m/の前「新聞」[ɕimbɯn]
  • [n]：/t, d, n/の前「案内」[annai]
  • [ŋ]：/k, g/の前「銀行」[ɡiŋkoː]
  • [ɴ]：語末や母音の前「本」[hoɴ]
  • 撥音の実現は同化規則による
  • 音節末子音（coda）

問題 次のうち、日本語で音素/r/と/l/が区別されない理由として正しいものはどれか。

  1. 日本語話者は両者を聞き分けられない
  2. 日本語では両者が最小対を作らない
  3. /r/と/l/は物理的に同じ音である
  4. 日本語に[r]も[l]も存在しない
  5. 日本語では音素の区別が存在しない

音素の判定基準：

  • 最小対の存在が必要
  • 日本語では[r]と[l]を入れ替えても意味が変わらない

英語の例

  • “right” [raɪt] vs “light” [laɪt]（最小対が存在） 英語では/r/と/l/は異なる音素
  • 言語によって音素体系が異なる
  • 第二言語習得の困難さの原因

問題 日本語で音節の「核」（nucleus）として機能するものはどれか。

  1. 子音のみ
  2. 母音
  3. 促音
  4. 撥音
  5. すべての音

音節の構造：

  • 頭子音（onset）：音節の始まり（任意）
  • 核（nucleus）：音節の中心（必須） すべての音節に必ず存在する
  • 尾子音（coda）：音節の終わり（任意）

核の特徴：

  • 通常は母音が核となる
  • 音節の中で最も響きが大きい
  • 核は音節の必須要素
  • 響度（sonority）が最も高い要素

問題 「連濁」に関する説明として正しいものはどれか。

  1. すべての複合語で起こる
  2. 複合語の後部要素の語頭が濁音化する現象
  3. 前部要素の語頭が濁音化する現象
  4. 外来語にのみ起こる
  5. 単語が単独で用いられるときに起こる現象

連濁：

  • 複合語の後部要素の語頭無声子音が有声化
  • 日本語の複合語形成に特徴的

連濁の制約：

  • ライマンの法則：後部要素に既に濁音がある場合、連濁しない
    • 例：「山」+「風」→「山かぜ」（*「山がぜ」）
  • 連濁は複雑な音韻現象
  • すべての複合語で起こるわけではない

問題 次のうち、日本語の音韻制約として正しいものはどれか。

  1. すべての子音連続が可能である
  2. 語頭に撥音「ん」は立たない
  3. 母音連続は不可能である
  4. すべての位置で促音が可能である
  5. 音韻制約は存在しない

日本語の音韻制約：

  • 語頭に撥音「ん」や促音「っ」は立たない
  • 複雑な子音連続は許されない
  • (C)(G)V(C)の構造 音節末に立てる子音は限られる（/N/, /Q/, /R/）

外来語での対応：

  • “strike” → 「ストライク」 （母音挿入で音節構造を調整）
  • 音韻制約は言語によって異なる
  • 外来語の適応でよく見られる

例：

  • ✗「んご」（語頭の撥音は不可）
  • ✓「りんご」（語中の撥音は可）

問題 次のうち、超分節音素（prosodic features）に含まれるものはどれか。

  1. /p/, /t/, /k/
  2. /a/, /i/, /u/
  3. アクセント、イントネーション、リズム
  4. 促音、撥音、長音
  5. 清音、濁音、半濁音

超分節音素（韻律的特徴）：

  • 音節や語を超えた音韻的特徴
  • 分節音素と対比される概念
  • 超分節音素は感情表現にも関わる
  • 失語症や自閉症で障害されることがある
